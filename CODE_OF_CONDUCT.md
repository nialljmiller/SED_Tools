# SED_Tools — Code of Conduct

## Scope

This Code of Conduct applies to all project spaces, including the repository, issue tracker, pull requests, and any communication channels associated with SED_Tools.

## Standards

Behavior that keeps this project functional and productive:

- Communicating clearly and respectfully
- Giving and receiving technical criticism constructively
- Staying on topic — this is a technical project

Unacceptable behavior:

- Harassment, personal attacks, or sustained hostility toward contributors
- Deliberate disruption of technical discussions
- Publishing others' private contact information without consent

## No Warranties

Participation in this project, including any contributions, advice, or code, comes with no guarantees of any kind. See the project license for full terms.

## License

This Code of Conduct, like the project itself, is fully open source and free for anyone to use, modify, or redistribute.

## Enforcement

Violations may be reported to **niall.j.miller@gmail.com**. Reports will be reviewed at the maintainer's discretion.

**1. Warning** — A private written warning explaining the violation and expected change in behavior.

**2. Ban** — Removal from the project and all associated spaces, temporary or permanent, depending on severity.

The maintainer has final say on all enforcement decisions.
